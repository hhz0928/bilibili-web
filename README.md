# Bilibili Clone 项目说明

这是一个仿B站首页布局的前端UI代码，使用HTML+CSS+JavaScript实现。

## 项目结构

```
bilibili-clone/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── script.js       # JavaScript脚本
├── images/
│   └── logo.svg        # Logo文件
└── README.md           # 项目说明文档
```

## 功能特性

1. **响应式布局**：
   - 顶部导航栏
   - 左侧分类侧边栏
   - 中间内容卡片列表
   - 右侧推荐栏

2. **交互功能**：
   - 移动端侧边栏可折叠
   - 导航菜单切换效果
   - 视频卡片悬停效果
   - 搜索、登录、注册按钮事件

3. **视觉效果**：
   - 浅蓝色主题配色
   - 卡片淡入动画
   - hover悬浮效果
   - 阴影效果

## 运行方式

1. 直接在浏览器中打开 `index.html` 文件即可预览效果
2. 或者使用本地服务器运行（推荐）

## 浏览器兼容性

支持所有现代浏览器（Chrome, Firefox, Safari, Edge等）

## 注意事项

- 图片资源使用了随机图片服务作为占位符
- 所有交互功能均为前端演示，实际项目中需要连接后端接口
- 响应式设计适配了不同屏幕尺寸
